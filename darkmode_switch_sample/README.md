# darkmode_switch_sample

アプリ内でダークモードへの切り替えを実装するサンプル。

状態管理にprovider、設定の保存にはshared_preferencesを用いている。

## 導入パッケージ

```yaml
provider: ^5.0.0
shared_preferences: ^2.0.7
```

※どちらもダークモードの対応に必須というわけではない
